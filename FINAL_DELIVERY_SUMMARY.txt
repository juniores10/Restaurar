EXTRACTION METHODOLOGY
================================================================================

CHALLENGE: Original XLS file was 0 bytes on disk but embedded in JSON transcript

SOLUTION STEPS:
1. Located XLS content embedded as Unicode text in JSON document
2. Parsed binary OLE2 format from raw Unicode strings
3. Extracted ASCII text portions using regex filtering
4. Applied pattern matching to identify employee records:
   - Pattern: NAME (ID) CPF CTPS PHONE EMAIL ... ADDRESS CITY ZIP
5. Used context-aware regex for each field extraction
6. Performed deduplication by CPF (418 unique records from 702 CPF patterns)
7. Exported to JSON and CSV formats

REGEX PATTERNS USED:
- Name: [A-Z][A-Z\s]+ followed by (digits)
- CPF: \d{3}\.\d{3}\.\d{3}-\d{2} or \d{11}
- Email: [\w\.-]+@[\w\.-]+\.\w+
- Phone: \d{4,5}-\d{4}
- Address: R|EST + text + \d{5}-\d{3}
- City: [A-Z\s]+-RJ pattern
